Introduction

This is basically a traffic usage monitor using vnStat (for measuring) and vnStati (for graphical design). This whole dashboard runs in Python using Flask and it requires almost no configuration.

How does it work

The way this works is by using the vnStati command to generate tables of network traffic usage so that you can have a more graphical vision of the normal vnStat database.

This even uses a automatic updater for the vnStati images that you can set up your own custom time to update.
